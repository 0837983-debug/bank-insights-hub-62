# Решение проблем

## Проблемы с подключением к БД

### Ошибка: "Connection refused"

**Причина:** База данных не запущена или недоступна.

**Решение:**
1. Проверьте, что PostgreSQL запущен
2. Проверьте параметры подключения в `backend/.env`
3. Проверьте доступность хоста: `ping your-db-host`

### Ошибка: "password authentication failed"

**Причина:** Неверные учетные данные.

**Решение:**
1. Проверьте `DB_USER` и `DB_PASSWORD` в `.env`
2. Убедитесь, что пользователь существует в БД

## Проблемы с миграциями

### Ошибка: "relation already exists"

**Причина:** Таблицы уже созданы.

**Решение:**
```bash
# Проверьте существующие таблицы
cd backend
npm run check-db-connection

# Если нужно пересоздать, сначала удалите схему
# (ОСТОРОЖНО: это удалит все данные!)
```

## Проблемы с запуском серверов

### Backend не запускается

**Проверьте:**
1. Установлены ли зависимости: `cd backend && npm install`
2. Правильно ли настроен `.env`
3. Свободен ли порт 3001: `lsof -i :3001`

### Frontend не запускается

**Проверьте:**
1. Установлены ли зависимости: `npm install`
2. Свободен ли порт 8080: `lsof -i :8080`
3. Запущен ли backend (frontend зависит от API)

## Проблемы с тестами

### Тесты не запускаются

**Решение:**
```bash
# Переустановите зависимости
npm install

# Проверьте конфигурацию
npm run test -- --help
```

### E2E тесты падают

**Причина:** Backend не запущен.

**Решение:**
1. Запустите backend: `cd backend && npm run dev`
2. Убедитесь, что он доступен на `http://localhost:3001`

## Проблемы с зависимостями

### Ошибка: "Cannot find module"

**Решение:**
```bash
# Переустановите зависимости
rm -rf node_modules package-lock.json
npm install

# Для backend
cd backend
rm -rf node_modules package-lock.json
npm install
```

## Проблемы с TypeScript

### Ошибки типов

**Решение:**
```bash
# Проверьте типы
npm run type-check

# Пересоберите типы
npm run build
```

## Получение помощи

Если проблема не решена:

1. Проверьте логи в консоли
2. Проверьте документацию в соответствующих разделах
3. Создайте issue с описанием проблемы
