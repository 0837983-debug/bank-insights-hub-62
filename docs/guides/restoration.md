# Инструкция по восстановлению файлов

## Если файл не найден

Если при работе с проектом вы столкнулись с отсутствием какого-либо файла, следуйте следующей процедуре:

### 1. Проверка списка неиспользуемых файлов

**Сначала проверьте наличие файла в списке неиспользуемых файлов:**
- Откройте файл `docs/unused-files.txt`
- Найдите путь к отсутствующему файлу в списке

### 2. Восстановление из архива

**Если файл найден в списке неиспользуемых файлов:**

1. Проверьте наличие файла в архиве:
   ```bash
   ls -la archive/[путь_к_файлу]
   ```

2. Восстановите файл из архива с сохранением структуры директорий:
   ```bash
   # Пример восстановления файла
   mkdir -p $(dirname "src/pages/Index.tsx")
   cp archive/src/pages/Index.tsx src/pages/Index.tsx
   ```

3. Или восстановите весь файл с полным путем:
   ```bash
   cp archive/src/pages/Index.tsx src/pages/Index.tsx
   ```

### 3. Создание нового файла

**Создавайте новый файл только в следующих случаях:**

- Файл **не найден** в списке `docs/unused-files.txt`
- Файл **не найден** в архиве `archive/`
- Попытка восстановления из архива **не удалась** (файл поврежден или отсутствует)

### 4. Обновление списка неиспользуемых файлов

После восстановления файла из архива:
- Если файл снова используется в проекте, удалите его из `docs/unused-files.txt`
- Если файл был восстановлен для временного использования, верните его в архив после завершения работы

## Структура архива

Архив сохраняет полную структуру директорий проекта:
```
archive/
├── src/
│   ├── pages/
│   ├── components/
│   ├── hooks/
│   ├── data/
│   ├── types/
│   ├── test/
│   └── lib/
├── scripts/
├── public/
└── ...
```

## Примеры восстановления

### Восстановление страницы:
```bash
cp archive/src/pages/Index.tsx src/pages/Index.tsx
```

### Восстановление компонента:
```bash
cp archive/src/components/report/BalanceSection.tsx src/components/report/BalanceSection.tsx
```

### Восстановление директории:
```bash
cp -r archive/playwright-report ./
```

## Важно

- **Всегда проверяйте архив перед созданием нового файла**
- **Сохраняйте структуру директорий при восстановлении**
- **Обновляйте список неиспользуемых файлов при изменении статуса файлов**
