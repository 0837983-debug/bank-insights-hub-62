# Prototype Branch (prototype/lovable)

Эта ветка используется для хранения прототипов и макетов, созданных в Lovable.

## Назначение

- **Разработка прототипов** - быстрая итерация и эксперименты
- **Визуальные макеты** - создание UI компонентов
- **Тестирование идей** - проверка концепций перед переносом в production

## Важно

⚠️ **Не используйте код из этой ветки напрямую в production!**

Все компоненты из прототипа должны быть:
1. Перенесены в ветку `main`
2. Адаптированы под структуру проекта
3. Протестированы и проверены

## Как использовать

1. Разрабатывайте прототипы в Lovable
2. Lovable автоматически коммитит код в эту ветку
3. Когда компонент готов - запросите перенос в `main` через Cursor

## Структура

Прототипы могут иметь любую структуру, удобную для Lovable.
При переносе в `main` структура будет адаптирована под стандарты проекта.
