# System Context

Этот раздел содержит актуальные снимки состояния системы. Используется для:

1. **Восстановления контекста** — агенты читают перед началом работы
2. **Быстрого понимания системы** — для новых участников
3. **Отслеживания изменений** — кто, что, когда менял

## Файлы контекста

| Файл | Обновляет | Содержимое |
|------|-----------|------------|
| [Backend](./backend.md) | Backend Agent | Сервисы, API, архитектура бэкенда |
| [Frontend](./frontend.md) | Frontend Agent | Компоненты, хуки, утилиты фронта |
| [Database](./database.md) | Backend Agent | Схемы, таблицы, views |

## Правила обновления

1. **После каждого изменения** — агент обновляет свой context-файл
2. **Формат** — краткие таблицы, списки, без лишнего текста
3. **Дата** — обновлять "Последнее обновление" в шапке файла

## Как использовать

### Для агентов (в инструкции):
```
Прочитай для восстановления контекста:
- docs/context/backend.md
- docs/context/database.md
```

### Для людей:
Открой в VitePress: `http://localhost:5173/context/`
